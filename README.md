# Tweejump
## tweejump介绍 ## 
tweejump 是cocos2d 编写的iphone游戏。通过晃动手机，看小鸟踏云能跳多高。

## 编译 ##
用xcode打开项目文件即可编译。

## 运行 ##
打包好后上传到iphone或ipod中运行。
